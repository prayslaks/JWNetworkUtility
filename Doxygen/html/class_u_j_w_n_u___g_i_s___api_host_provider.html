Detailed Description

DefaultJWNetworkUtility.ini 설정값을 CPP와 블루프린트로 읽을 수 있도록 돕는 게임인스턴스 서브시스템. 서비스 타입별로 다른 호스트 URL을 TMap으로 관리한다.
