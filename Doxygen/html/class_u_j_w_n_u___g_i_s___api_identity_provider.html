Detailed Description

인증 JWT 엑세스 토큰, 리프레시 토큰, UserId를 관리하는 게임인스턴스 서브시스템. 서비스 타입별로 다른 엑세스 토큰을 TMap으로 관리하며, 리프레시 토큰을 암호화-복호화한다. UserId는 단일 FString(메모리 전용, 로그인/리프레시 응답에서 수신)으로 관리한다.
